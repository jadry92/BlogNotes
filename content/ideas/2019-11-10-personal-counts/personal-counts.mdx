This idea is to create a personal web application where you can see
all your incomes and spends to help you in the next tax declaration.
Also, web application allows creating a saving planning for goals.

This project will be made in Django with the goal of all your data will
secure and encrypted for your personal use.

The repository for this project is [here](https://github.com/jadry92/Tax-assistant).
