REGEX or Regular Expressions are the key tool that all Developer shoul has

```
. -> whatever ca racter

\d - > find all the digits
\s - > spaces in blank
[0-9] ~ \d
[0-9a-zA-Z_] ~ \w
.* -> greedy all
+ -> one or more
? -> zero or one
```
